# 游戏音频文件

## 所需音频文件

请将以下音频文件放在此目录下：

1. **background.mp3** - 背景音乐
   - 建议：轻快、愉悦的音乐
   - 时长：2-3分钟（会循环播放）
   - 音量：中等

2. **slice.mp3** - 切割音效
   - 建议：清脆的切割声音
   - 时长：0.5-1秒
   - 音量：较高

3. **perfect.mp3** - 完美切割音效
   - 建议：成功、庆祝的音效
   - 时长：1-2秒
   - 音量：较高

## 音频格式

- 支持格式：MP3, WAV, OGG
- 推荐格式：MP3（兼容性最好）
- 推荐码率：128-192 kbps

## 备注

如果没有音频文件，游戏仍可正常运行，只是没有声音。游戏会自动检测音频文件是否存在。

某些浏览器需要用户交互（如点击按钮）后才能播放音频。这是正常的安全限制。
