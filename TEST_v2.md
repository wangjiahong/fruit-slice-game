## 🎯 5个关键测试（2分钟）

### ✅ 测试1：刀的方向（10秒）
1. 打开游戏
2. 点击"开始游戏"
3. 鼠标移到画布上

**期望**：
- 看到🔪emoji光标
- **刀尖指向右下方**（不是左上）
- **刀把在左上方**

### ✅ 测试2：无声音（5秒）
1. 开始游戏
2. 切割水果

**期望**：
- 没有任何声音
- 右上角没有音频按钮

### ✅ 测试3：线从刀尖出来（15秒）
1. 在画布上点击（不要移动）
2. 观察红色圆点位置
3. 慢慢拖动鼠标画一条线

**期望**：
- 红色圆点出现在点击位置
- 黑色线从红点开始延伸
- 线条跟随鼠标轨迹

### ✅ 测试4：精确评分（30秒）
1. 尝试精确切割西瓜（50:50）
2. 查看结果页面

**期望**：
- 偏差显示2位小数（如 0.23%）
- 分数显示1位小数（如 95.5）
- 左侧和右侧也是2位小数

### ✅ 测试5：真实排行榜（30秒）
1. 开始游戏
2. 查看右侧实时排名
3. 观察30秒

**期望**：
- 看到4-6个玩家
- 其他玩家分数在你的±50分内
- 分数偶尔小幅变化（±5到±15）
- 不会有人突然涨100分

## 🎨 视觉快速检查

### 光标检查
把鼠标移到画布上，看到的应该是：
```
     🔪  ← 刀把在这里
      \
       \  ← 刀尖指向这里（右下）
```

### 切割线检查
画线时应该看到：
```
🔴 ← 红色起点（你点击的地方）
 \
  \  ← 黑色线条
   \
    ● ← 当前鼠标位置（小黑点）
```

### 结果页面检查
应该显示：
```
左侧: 49.87%  ← 2位小数
右侧: 50.13%  ← 2位小数
偏差: 0.13%   ← 2位小数
得分: 99.4    ← 1位小数
```

## 🐛 常见问题

### Q: 刀尖还是朝错了？
**A**: 刷新页面（Ctrl+R 或 Cmd+R）

### Q: 还是有声音？
**A**:
- 检查是否有其他标签页在播放
- 刷新页面

### Q: 排行榜分数还是太离谱？
**A**:
- 刷新页面
- 应该看到分数差距在±50以内

### Q: 分数不显示小数？
**A**:
- 刷新浏览器缓存（Ctrl+Shift+R）
- 或者清空缓存后重新加载
