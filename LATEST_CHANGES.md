## 刀的旋转说明

### 旋转角度计算
```
0度   = 刀尖向右 →
90度  = 刀尖向下 ↓
180度 = 刀尖向左 ←
270度 = 刀尖向上 ↑

当前：270度（-90度）
```

### CSS transform
```css
transform="rotate(270 20 20)"
```
- 旋转270度
- 旋转中心：(20, 20)

## 如果需要调整

### 改变刀的方向
编辑 `css/style.css`，修改 rotate 值：
```css
rotate(0)    # 刀尖向右
rotate(90)   # 刀尖向下
rotate(180)  # 刀尖向左
rotate(270)  # 刀尖向上
rotate(-45)  # 刀尖向右上
rotate(45)   # 刀尖向右下
rotate(135)  # 刀尖向左下
rotate(225)  # 刀尖向左上
```

### 改变分数小数位数
编辑 `js/leaderboard.js`，修改 toFixed 参数：
```javascript
toFixed(0)  # 无小数
toFixed(1)  # 1位小数
toFixed(2)  # 2位小数
```
